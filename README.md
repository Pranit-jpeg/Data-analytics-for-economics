# Applied Data Analytics in Economics
Coursework from **ECON5200 – Applied Data Analytics in Economics** at Northeastern University.
  Projects span the full analytics pipeline — from data collection and EDA to regression,
  machine learning, causal inference, and structural demand estimation.

  ## Projects

  | Folder | Topic |
  |---|---|
  | `tiktok-claims-detection-eda` | EDA and predictive modeling for video claims detection (PACE workflow) |
  | `sql-excel-data-analysis` | Data wrangling and analysis using SQL and Excel |
  | `web-scraping-starbucks-global-prices` | Web scraping with BeautifulSoup and Selenium |
  | `statistical-foundations` | Sampling, distributions, confidence intervals, hypothesis testing |
  | `regression-analysis` | Simple linear, multiple linear, and logistic regression |
  | `machine-learning-fundamentals` | KMeans, Naive Bayes, Decision Trees, feature engineering |
  | `time-series-forecasting` | US Retail Sales and GDP forecasting with ML models |
  | `causal-inference-replications` | Replications of Card (1990), Dube et al., and Thistlethwaite-Campbell RDD |
  | `demand-estimation-io` | BLP, Nevo cereal demand, nested logit, random utility models |

  ## Stack
  Python · SQL · Excel · scikit-learn · pandas · NumPy · Matplotlib · BeautifulSoup · Selenium
